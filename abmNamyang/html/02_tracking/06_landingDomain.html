랜딩 도메인 생성

랜딩 도메인 생성이란? 광고추적의 성과를 극대화 하기 위해 필요한 랜딩페이지를 모비튠 트래킹에서 제공해주고 있습니다.

랜딩 도메인을 사용 시, 앱이 이미 설치된 경우 dep plink를depplink를 통해서 앱을 실행 시킬 수 있고, 앱이 없는 경우 마켓으로 이동 시켜 앱 설치에 기여 할 수 있습니다.

https://tracker.mobon.net/

* 사용방법 : 모비튠 랜딩 도메인을 복사 후 UTM 태그를 입력하여 원하는 이벤트/버튼에 삽입 하여 광고 추적을 경험 해보세요.

예시) https://tracker.mobon.net/광고주ID?utm_source=kakao&utm_medium=cpc&utm_campaign=event campaign

랜딩경로 지정

Android

앱 연결

Deeplink Schema

* 앱이 이미 설치된 경우, 특정 광고 및 이벤트를 클릭했을 때 Deeplink Schema 값을 통해서 해당 앱이 실행(Deeplink)됩니다.

Google
Play Store

* 앱이 설치되어 있지 않은 경우, 특정 광고 및 이벤트를 클릭 했을때 Package Name값을 통해서 Google Play Store 이동시켜 앱 설치를 기여합니다.

IOS

앱 연결

Deeplink Schema

* 앱이 이미 설치된 경우, 특정 광고 및 이벤트를 클릭했을 때 Deeplink Schema 값을 통해서 해당 앱이 실행(Deeplink)됩니다.

App Store

* 앱이 설치되어 있지 않은 경우, 특정 광고 및 이벤트를 클릭 했을 때 App Store ID 값을 통해서 Apple App Store 이동시켜 앱 설치를 기여합니다.

PC

PC URL

* PC에서 선택 시, 이동 시킬 URL을 입력해주세요.
