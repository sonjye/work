광고 추적 URL

필수 항목 (*표시)를 입력하면 광고 추천 URL이 생성됩니다.

생성 된 URL은 변경 시 자동으로 업데이트 됩니다.

Website URL

전체 웹사이트 URL (ex_http://example.com)

utm_source (소스)

참조자 (ex_google, newsletter) 어느 출처에서 오는지 입력

utm_medium (매체)

마케팅 매체 (ex_cpc, banner, email) 어느 매체를 통해서 오는지 입력

utm_campaign (캠페인)

제품, 프로모션 코드 또는 슬로건 (ex_spring_sale) 어느 매체를 통해서 오는지 입력

utm_term (키워드)

검색광고 시 사용되는 검색어를 입력

utm_content (광고콘텐츠)

동일한 광고 캠페인에서 소재가 여러 개일 경우, 이를 구분하기 위해 입력

광고 추적 URL 미리보기

생성된 광고 추천 URL을 트래킹 하고 싶은 버튼 및 링크에 삽입시키면, 광고 추적 보고서를 통해 데이터를 확인 하실 수 있습니다.
